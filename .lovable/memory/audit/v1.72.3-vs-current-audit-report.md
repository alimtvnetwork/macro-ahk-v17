## Critical Bugs Fixed

### 1. Variable Name Mismatch — `injection-handler.ts`
- **Root cause:** Code referenced `batchable` but the variable was renamed to `orderedScripts`.
- **Fix:** Updated reference to `orderedScripts`.

### 2. Missing `codeSource` Property — `PreparedInjectionScript`
- **Root cause:** Interface required `codeSource` but it was never populated during script preparation.
- **Fix:** Added `codeSource` assignment in the preparation pipeline.

### 3. `BgLogTag` Enum Inlining (Systemic)
- **Root cause:** 31 background modules used `BgLogTag` without importing it, relying on implicit availability that broke at runtime.
- **Fix:** Added explicit `import { BgLogTag }` to all 31 affected modules.

## Test Failures Resolved (18 files, ~48 tests)

### Category 1: Project Seed Over-Counting (5 files)
- **Root cause:** `ensureDefaultProjectSingleScript` created extra seed data, inflating project/script counts.
- **Fix:** Mocked `ensureDefaultProjectSingleScript` to no-op in affected test files.

### Category 2: Hot-Reload Mock Reset (2 files)
- **Root cause:** `chrome.runtime.getURL` mock not reset between tests, causing stale return values.
- **Fix:** Added proper mock cleanup in `beforeEach`.

### Category 3: Valid JWT for `GET_TOKEN` (3 files)
- **Root cause:** Tests used placeholder strings that failed JWT validation added in current code.
- **Fix:** Updated to properly structured JWT tokens (`header.payload.signature`).

### Category 4: Behavioral Changes (5 files)
- **Root cause:** Current code introduced new behaviors (script sorting by order, empty-script skipping, injection diagnostics).
- **Fix:** Updated test expectations to match current behavior.

### Category 5: Panel-Builder Auth Mock (1 file)
- **Root cause:** Missing `getAuthToken` in dependency mock.
- **Fix:** Added mock function.

### Category 6: Snapshot Regeneration (2 files)
- **Root cause:** Popup snapshot test missing `opfsStatus` and `debugMode` in `usePopupData` mock.
- **Fix:** Added missing mock fields and regenerated snapshot.
